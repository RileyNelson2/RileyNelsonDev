## WHAT I DO

**- ✅ Geospatial Application Development.**
I have hands-on experience building web applications that integrate spatial datasets, real-time sensor data, and interactive maps using React, TypeScript, Leaflet, and D3.js.

**- ✅ Full-Stack Web Development.**
From front-end interfaces to back-end databases, I build and maintain software across the stack using modern tools and frameworks in collaborative agile teams.
